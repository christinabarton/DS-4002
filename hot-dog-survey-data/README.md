# Hot Dog Survey Data

We collected data from full time undergraduate students by having them filling out a survey. The raw data are uploaded in the file Hotdog Survey (Responses).xlsx, and the data dictionary describing the data is in the file hotdog_data_dictionary.xlsx.

* Data Dictionary:
